# Mini Distributed Database

## Overview

Mini Distributed Database is a project to build a distributed key-value database from scratch using Modern C++.

The primary goal of this project is to understand how databases and distributed systems work internally by implementing each component step by step.

This project focuses on learning system programming concepts rather than using existing database libraries.

## Objectives

- Learn Modern C++
- Learn system programming
- Understand database internals
- Understand networking
- Learn concurrent programming
- Build a distributed system from scratch
- Practice software architecture
- Improve problem-solving skills

## Technology

- C++20
- CMake
- Git
- Linux (recommended)
